# Smart Mobility Simulator v1.0

## Project Overview
Technical demonstration of **Object-Oriented Programming (OOP)** developed for the UNAD University. This system simulates a vehicle fleet where each unit responds uniquely to global commands through advanced logic.

## Key Implementations
- **Inheritance:** Hierarchical structure based on a parent `Vehiculo` class.
- **Polymorphism:** Unique acceleration and braking behaviors for Electric Cars, Motorcycles, and Trucks.
- **Encapsulation:** Internal state management for speed and vehicle specifications.
- **GUI:** Professional interactive dashboard built with **Tkinter**.

## Deployment
1. Clone the repo: `git clone https://github.com/Julian732-flow/smart-mobility-simulator-py.git`
2. Run: `python vehicle_system.py`
